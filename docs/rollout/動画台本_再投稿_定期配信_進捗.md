## 1. 再投稿（約2分）

**画面の流れ**  
ホーム → **再投稿** → 一覧から1件選ぶ → **再利用して作成** → 新規投稿画面（内容確認）→ 期限を選ぶ → **送信**

**台本**

> ホームの **再投稿** を開きます。  
> ここには、過去に **新規投稿** で送った依頼だけが一覧に出ます。  
> **定期配信** で登録したものは、ここには表示されません。一覧は別です。  
>
> もう一度送りたい依頼の **再利用して作成** を押すと、新規投稿の画面に移ります。  
> 依頼内容・URL・添付・配信先は、そのときの設定が引き継がれます。  
> **期限だけは毎回選び直してください。**  
>
> 内容を確認して **送信** すれば、新規投稿と同じように配信され、対象者にメールが届きます。  
>
> 「先月と同じ内容を手動でもう一度送りたい」ときに使います。  
> 毎月自動で送りたい場合は、**定期配信** を使ってください。

**撮影メモ**

- [ ] 一覧に過去の新規投稿が1件以上ある状態で撮る
- [ ] 「再利用して作成」→ 新規投稿画面に遷移するところを見せる
- [ ] 期限を変更しているところを見せる
- [ ] 送信はテスト用依頼でOK（本番全体配信は避ける）

## 2. 定期配信（約3分）

**画面の流れ**  
ホーム → **定期配信** → フォーム入力（種類・スケジュール・内容・配信先）→ 保存 → **稼働中の定期配信** 一覧 → 編集 or 削除

**台本**

> ホームの **定期配信** では、毎月決まった日に同じ To Do を **自動で送る** 設定ができます。  
>
> まず **依頼の種類** を選びます。社員・店舗・TFチームの3種類です。  
> 次に **配信スケジュール** を設定します。  
> **毎月何日に配信するか** を選びます。配信時刻は **午前10時** 固定です。  
> タスクの期限は、**月末まで** または **○日まで** を選べます。  
>
> 新規登録のとき、「**今月の初回分は作成しない**」にチェックを入れると、  
> 翌月の配信日から自動配信が始まります。  
>
> 依頼内容と配信先は、新規投稿と同じ要領で設定して **保存** します。  
>
> 画面下の **稼働中の定期配信** 一覧から、内容の **編集** や **削除（停止）** もできます。  
> 編集した内容は、**次回の定期配信から** 反映されます。今日すぐタスクが増えるわけではありません。  
>
> **再投稿** は手動でもう一度送る機能です。**定期配信** は毎月自動で送る設定です。混同しないようご注意ください。

**撮影メモ**

- [ ] 配信日・期限のドロップダウンを操作
- [ ] 「今月の初回分は作成しない」チェックを見せる
- [ ] 稼働中の一覧（あれば編集画面）を見せる
- [ ] 実保存はテスト用でOK
