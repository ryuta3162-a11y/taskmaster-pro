# 動画台本：再投稿・定期配信・チーム進捗・admin

ガイドサイト（`guide-site/index.html`）の文面と揃えた収録用台本です。  
**各章2〜3分**を目安に、画面操作しながら読み上げてください。

---

## 撮影の順番（おすすめ）

1. **再投稿**（依頼を出す人向け・約2分）
2. **定期配信**（依頼を出す人向け・約3分）
3. **チーム進捗管理ダッシュボード**（リーダー・TF向け・約2分）
4. **進捗管理admin**（DX・管理者向け・約2分）

※ 実送信が不安な章は、一覧表示まででもOK（再投稿はテスト用依頼で送信可）。

---

## 1. 再投稿（約2分）

**画面の流れ**  
ホーム → **再投稿** → 一覧から1件選ぶ → **再利用して作成** → 新規投稿画面（内容確認）→ 期限を選ぶ → **送信**

**台本**

> ホームの **再投稿** を開きます。  
> ここには、過去に **新規投稿** で送った依頼だけが一覧に出ます。  
> **定期配信** で登録したものは、ここには表示されません。一覧は別です。  
>
> もう一度送りたい依頼の **再利用して作成** を押すと、新規投稿の画面に移ります。  
> 依頼内容・URL・添付・配信先は、そのときの設定が引き継がれます。  
> **期限だけは毎回選び直してください。**  
>
> 内容を確認して **送信** すれば、新規投稿と同じように配信され、対象者にメールが届きます。  
>
> 「先月と同じ内容を手動でもう一度送りたい」ときに使います。  
> 毎月自動で送りたい場合は、**定期配信** を使ってください。

**撮影メモ**

- [ ] 一覧に過去の新規投稿が1件以上ある状態で撮る
- [ ] 「再利用して作成」→ 新規投稿画面に遷移するところを見せる
- [ ] 期限を変更しているところを見せる
- [ ] 送信はテスト用依頼でOK（本番全体配信は避ける）

## 2. 定期配信（約3分）

**画面の流れ**  
ホーム → **定期配信** → フォーム入力（種類・スケジュール・内容・配信先）→ 保存 → **稼働中の定期配信** 一覧 → 編集 or 削除

**台本**

> ホームの **定期配信** では、毎月決まった日に同じ To Do を **自動で送る** 設定ができます。  
>
> まず **依頼の種類** を選びます。社員・店舗・TFチームの3種類です。  
> 次に **配信スケジュール** を設定します。  
> **毎月何日に配信するか** を選びます。配信時刻は **午前10時** 固定です。  
> タスクの期限は、**月末まで** または **○日まで** を選べます。  
>
> 新規登録のとき、「**今月の初回分は作成しない**」にチェックを入れると、  
> 翌月の配信日から自動配信が始まります。  
>
> 依頼内容と配信先は、新規投稿と同じ要領で設定して **保存** します。  
>
> 画面下の **稼働中の定期配信** 一覧から、内容の **編集** や **削除（停止）** もできます。  
> 編集した内容は、**次回の定期配信から** 反映されます。今日すぐタスクが増えるわけではありません。  
>
> **再投稿** は手動でもう一度送る機能です。**定期配信** は毎月自動で送る設定です。混同しないようご注意ください。

**撮影メモ**

- [ ] 配信日・期限のドロップダウンを操作
- [ ] 「今月の初回分は作成しない」チェックを見せる
- [ ] 稼働中の一覧（あれば編集画面）を見せる
- [ ] 実保存はテスト用でOK

## 3. チーム進捗管理ダッシュボード（約2分）

**画面の流れ**  
ホーム下部 → **〇〇チーム進捗管理ダッシュボード** → 進捗画面（閲覧のみ）

**台本**

> ホーム画面の下に、自分が所属するチーム名の **進捗管理ダッシュボード** があります。  
> ここでは、そのチームに関係する To Do の **進捗状況を見るだけ** の画面です。  
>
> 依頼ごとに、期限や進捗の状況を確認できます。  
> **完了操作はできません。** あくまで閲覧用です。  
>
> チームリーダーや TF の方が、「全体の進み具合」を確認するときに使います。  
> 所属チームが複数ある方は、チームごとにボタンが表示されます。  
>
> 未完了の人にメールで促すのは、DX など管理者向けの **進捗管理admin** です。役割が違います。

**撮影メモ**

- [ ] ホーム下部のダッシュボードボタンをタップ
- [ ] 進捗一覧・カードが見える状態
- [ ] 「見るだけ」と一言添える

## 4. 進捗管理admin（約2分）

**画面の流れ**  
ホーム → **進捗管理admin**（表示される人のみ）  
または `?page=admin` のURL → 社員依頼 / 店舗依頼 / 定期配信タブ → 依頼を展開 → リマインド（デモで確認ダイアログまででも可）

**台本**

> **進捗管理admin** は、DX チームなど **管理者だけ** が使う画面です。  
> ホームにこのメニューが出る方だけがアクセスできます。一般の方は使いません。  
>
> 画面上部のタブで、**社員依頼**・**店舗依頼**・**定期配信** を切り替えられます。  
> 社員依頼では未完了の方を、店舗依頼では未完了の店舗を確認できます。  
> 定期配信タブでは、登録されている定期配信の一覧を参照できます。  
>
> 依頼の行をタップして展開し、未完了の人や店舗を選んで、  
> **リマインドメール** を送ることができます。  
>
> 管理者用の URL は、利用者アプリの URL の末尾に **?page=admin** を付けたものです。  
> ブックマークしておくと便利です。

**撮影メモ**

- [ ] 3つのタブを切り替え
- [ ] 依頼1件を展開して未完了一覧を見せる
- [ ] リマインドは実送信せず確認ダイアログまででも可
- [ ] 一般ユーザーにはこの画面が出ない旨を口頭で補足
