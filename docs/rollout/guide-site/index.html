To Do List の使い方

テキストと動画で、依頼の出し方・受け方・進捗の見方をかんたんにご案内します。 まずはリストチェック、依頼を出す方は新規投稿をご覧ください。

ログイン・初回登録

動画 約2分

解説動画

社内メールアドレスでログインし、プロフィールを登録します。

  • チーム・役職は必ず選択してください。
  • 店舗エリア所属の方は 第1〜7エリア と 管轄店舗 を選びます。EAST本部は選択不要です。
  • EAST本部所属の方は、エリアの代わりに EAST本部 を選べば、エリア・店舗の選択は不要です。

管轄店舗について

管理したい店舗を多く選択いただいても構いません。エリア・テリトリー内で社員間で相談し、誰がどの店舗のタスク管理をするかを決めてから登録してください。

登録後は「ログイン情報の変更」からいつでも更新できます。

リストチェック

動画 約4分 ／ 全員必須

解説動画

依頼を受け取った方は、ホームのリストチェックを開きます。 配信が完了すると 通知メール が届き、メール内のリンクからも同じ画面を開けます。 ホームのバッジ数字は未完了件数です。

依頼の種類 社員依頼・店舗依頼・TFチーム依頼で絞り込みできます。
店舗依頼 担当店舗ごとに完了します。画面上部の店舗名チップで絞り込み、店舗単位で完了操作ができます。

新規投稿（To Do 申請）

動画 約4分 ／ 依頼を出す方

解説動画

ホーム画面の 新規投稿 をタップして、依頼を作成します。まず 依頼の種類 を選びます。

社員への依頼 役職別で配信できます（社員へ依頼したい場合）。該当する役職の方が、それぞれリストチェックで完了します。
店舗への依頼 エリア・店舗別で配信できます（店舗へ依頼したい場合）。担当店舗ごとに完了します。
TFチームの依頼 所属チーム別に配信できます。社員への依頼と同じタスク配信形式のため、各チームで1名が完了しても、チーム全体の完了にはなりません（該当チームのメンバーそれぞれが完了が必要です）。

依頼内容・期限を入力し、配信先を確認して 送信 します。 配信が完了すると メールが届き、受信者の リストチェック に To Do が表示されます。

Gmail でラベルを付ける（おすすめ）

件名で自動振り分け ／ 約2分

To Do の通知メールは件名が 【TODOリスト】新しいタスクが届きました で統一されています。 Gmail のフィルタで To-Do List ラベルを自動で付けると、受信トレイが整理されて管理しやすくなります。

  1. Gmail を開き、検索ボックスに次を入力
    subject:【TODOリスト】新しいタスクが届きました
  2. 検索オプション（フィルタ作成）を開く →「この検索条件で検索」で一度確認してもOK
  3. 「フィルタを作成」 をクリック
  4. 「ラベルを付ける」 にチェック → ラベルで To-Do List を選択 （なければ「新規作成」で To-Do List を作ります）
  5. 必要なら「○件の一致するスレッドにもフィルタを適用する」にチェック（過去の通知もまとめられます）
  6. 「フィルタを作成」 で完了

設定後のイメージ

件名が 【TODOリスト】新しいタスクが届きました のメールに、自動で To-Do List ラベルが付きます。左メニューから To-Do 通知だけを一覧できます。

再投稿

動画 約2分 ／ 依頼を出す方

解説動画

ホームの 再投稿 を開くと、過去に 新規投稿 で送った依頼だけが一覧に出ます。 定期配信で登録したものはここには表示されません（一覧は別です）。

一覧から選ぶ 送りたい依頼の 再利用して作成 を押します。
引き継がれるもの 依頼内容・URL・添付ファイル・配信先（役職・店舗・チームなど）。
選び直すもの 期限 は毎回選び直してください。
送信 内容を確認して 送信 すれば、新規投稿と同様に配信・通知メールが届きます。

こんなときに使う

「先月と同じ内容を、もう一度手動で送りたい」「内容はほぼ同じで期限だけ変えたい」といったときに便利です。毎月自動で送りたい場合は 定期配信 を使います。

定期配信

動画 約3分 ／ 依頼を出す方

解説動画

ホームの 定期配信 では、毎月決まった日に同じ To Do を自動で送る設定ができます。 新規投稿と同様に、依頼の種類・内容・配信先を設定します。

配信日 毎月何日に配信するか を選びます。配信時刻は 午前10:00 固定です。
期限 毎月のタスク期限（月末 または ○日まで）を選びます。
今月は送らない 新規登録時、「今月の初回分は作成しない」にチェックすると、翌月の配信日から開始できます。
編集・停止 下部の 稼働中の定期配信 一覧から編集・削除（停止）できます。編集内容は次回の定期配信から反映されます。

再投稿との違い

再投稿は手動でもう一度送る機能。定期配信は毎月自動で送る設定です。一覧も別なので、混同しないようご注意ください。

チーム進捗管理ダッシュボード

動画 約2分 ／ チームリーダー・TF向け

解説動画

ホーム画面の下部にある 〇〇チーム進捗管理ダッシュボード から、 そのチームの TF 依頼などの進捗を閲覧できます。

できること 依頼ごとの進捗状況・期限・完了率を確認できます。
できないこと 完了操作はできません（見るだけの画面です）。
複数チーム 所属チームが複数ある方は、チームごとにボタンが表示されます。

管理者画面との違い

チームリーダー向けはこの進捗ダッシュボード。DX などが未完了者にメールを送るのは 進捗管理admin です。役割が違います。

進捗管理admin（管理者向け）

動画 約2分 ／ DX・管理者のみ

解説動画

進捗管理admin は、DX チームなど管理者だけが使う画面です。 ホームに表示される方のみアクセスできます（一般の方は利用しません）。

社員依頼 社員向け依頼の進捗一覧。未完了の方を確認できます。
店舗依頼 店舗向け依頼の進捗一覧。未完了の店舗を確認できます。
定期配信 登録されている定期配信の一覧（参照のみ）。
リマインド 依頼を展開し、未完了の人（または店舗）を選んで リマインドメール を送れます。

開き方（管理者用）

管理者用 URL は利用者アプリの URL の末尾に ?page=admin を付けたものです。ブックマークしておくと便利です。

よくある質問

Gmail で To-Do 通知だけ見たい

件名 【TODOリスト】新しいタスクが届きました でフィルタを作り、ラベル To-Do List を付けてください。設定手順はこちら。

テリトリー4や管轄店舗が出ない

エリアとテリトリーを正しく選んでください。第4エリアのみテリトリー4があります。解決しない場合は一時登録後、「ログイン情報の変更」で修正できます。

EAST本部所属はどう登録する？

登録画面で「EAST本部」を選びます。エリア・店舗の選択は不要です。店舗依頼のエリア配信には含まれませんが、社員依頼・TF依頼は受け取れます。

再投稿と定期配信の違いは？

再投稿は過去の新規投稿を手動でもう一度送る機能。定期配信は毎月自動で送る設定です。一覧も別です。

動画が再生されない

MP4 を使う場合は videos/ にファイルがあるか、guide-config.js のパスを確認してください。ブラウザはタップ後の再生に対応しています（自動再生は音ありでは制限されます）。
