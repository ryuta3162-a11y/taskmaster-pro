# 各章のナレーション長（収録の目安）

自動音声を生成済みです（`videos/audio/*-narration.mp3`）。  
画面収録は **この秒数＋α** を目安に、ゆっくり操作してください。

| 章 | 秒数 | 目安 | 収録ファイル名 | 合成コマンド |
|----|------|------|----------------|--------------|
| overview | 52秒 | 約1分 | `overview-screen.mp4` | `.\merge_narration.ps1 overview` |
| register | 44秒 | 約1分 | `register-screen.mp4` | `.\merge_narration.ps1 register` |
| checklist | 70秒 | 約1.5分 | `checklist-screen.mp4` | `.\merge_narration.ps1 checklist` |
| request | 59秒 | 約1.5分 | `request-screen.mp4` | `.\merge_narration.ps1 request` |
| repost | 39秒 | 約1分 | `repost-screen.mp4` | `.\merge_narration.ps1 repost` |
| scheduled | 46秒 | 約1分 | `scheduled-screen.mp4` | `.\merge_narration.ps1 scheduled` |
| progress | 36秒 | 約1分 | `progress-screen.mp4` | `.\merge_narration.ps1 progress` |
| gmail-label | 36秒 | 任意 | `gmail-label-screen.mp4` | `.\merge_narration.ps1 gmail-label` |

詳細な同期表は各章の `*-narration.vtt` を参照してください。

## 音声の聞き方

エクスプローラーで `docs/rollout/guide-site/videos/audio/` を開き、  
`overview-narration.mp3` などをダブルクリックで再生できます。

## 再生成

台本を直したあと:

```powershell
cd docs\rollout\guide-site\scripts
python generate_narration.py overview
```
