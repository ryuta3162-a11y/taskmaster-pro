## 収録のコツ

1. 先に `overview-narration.mp3` を再生して1回リハーサル
2. OBS や Xbox Game Bar で **システム音声は録らない**（ナレーションは後から合成）
3. スマホ利用が多いなら、スマホ幅で収録

## 収録後（合成）

```powershell
cd docs\rollout\guide-site\scripts
.\merge_narration.ps1 overview
```

→ `videos/overview.mp4` と `videos/overview.vtt` ができます。
