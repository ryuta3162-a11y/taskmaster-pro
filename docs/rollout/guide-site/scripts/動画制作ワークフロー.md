# 動画制作ワークフロー（v2・自動音声付き）

画面収録は **無音**、ナレーションは **自動音声** で後から合成する手順です。

---

## 全体の流れ

```
1. 台本確認（動画台本_v2_全章.md）
2. 画面収録（無音・MP4）
3. 自動音声生成（generate_narration.py → MP3 + VTT）
4. 動画と音声を合成（ffmpeg）
5. videos/ に配置 → guide-config.js 更新 → Vercel 再デプロイ
```

---

## 1. 準備（初回のみ）

### Python と edge-tts

社用PCで Python が使える場合:

```powershell
cd docs\rollout\guide-site\scripts
pip install -r requirements-narration.txt
```

### ffmpeg（音声合成用）

- [ffmpeg](https://ffmpeg.org/download.html) をインストールし、PATH に通す
- 確認: `ffmpeg -version`

---

## 2. 画面収録

| 項目 | 推奨 |
|------|------|
| ツール | OBS、Xbox Game Bar、PowerPoint の画面収録など |
| 音声 | **オフ**（マイク無効） |
| 解像度 | 実際の利用端末に合わせる（スマホ利用ならスマホ幅） |
| ファイル名 | `checklist-screen.mp4` など（章ごと） |

台本は `docs/rollout/動画台本_v2_全章.md` を見ながら、**ナレーションのペースに合うようゆっくり操作** してください。

---

## 3. 自動音声の生成

```powershell
cd docs\rollout\guide-site\scripts

# 全章まとめて
python generate_narration.py

# 1章だけ（例: リストチェック）
python generate_narration.py checklist
```

### 出力先

```
docs/rollout/guide-site/videos/audio/
  overview-narration.mp3
  overview-narration.vtt
  checklist-narration.mp3
  checklist-narration.vtt
  ...
```

### 音声の調整

`scripts/narration/*.json` を編集:

| キー | 例 | 説明 |
|------|-----|------|
| `voice` | `ja-JP-NanamiNeural` | 女性（標準） |
| `voice` | `ja-JP-KeitaNeural` | 男性 |
| `rate` | `-10%` | ゆっくり |
| `rate` | `+5%` | 少し速く |
| `segments[].text` | 台本1文 | 読み上げ文 |

利用可能な声一覧:

```powershell
edge-tts --list-voices | findstr ja-JP
```

## 4. 動画と音声の合成（ffmpeg）

画面収録とナレーションの長さがずれる場合は、先にどちらかを基準に編集してください。

### 基本（音声を動画の長さに合わせる）

```powershell
cd docs\rollout\guide-site\videos

ffmpeg -y -i checklist-screen.mp4 -i audio/checklist-narration.mp3 `
  -c:v copy -c:a aac -b:a 192k `
  -map 0:v:0 -map 1:a:0 -shortest `
  checklist.mp4
```

- `-shortest` … 短い方に合わせて切る
- 完成ファイル `checklist.mp4` を `videos/` に置く

### 音声が長い場合（動画を伸ばす・最後に静止画）

音声に合わせたいときは、編集ソフトで画面録画の末尾を伸ばすか、ffmpeg で `tpad` を使います（上級者向け）。

### 字幕ファイル

ガイドサイトの字幕には、合成後に次のどちらかを使います。

- **A.** `audio/checklist-narration.vtt`（自動音声と同期）
- **B.** 手直しした `videos/checklist.vtt`

## 5. ガイドサイトへの反映

### guide-config.js

```js
checklist: {
  title: 'リストチェック',
  duration: '約4分',
  mp4: 'videos/checklist.mp4',
  poster: 'images/checklist-poster.png',
  vtt: 'videos/checklist.vtt',  // または audio/checklist-narration.vtt
  ...
},
```

新規章（全体の流れ）を追加する例:

```js
overview: {
  title: '全体の流れ（はじめに）',
  duration: '約3分',
  mp4: 'videos/overview.mp4',
  poster: '',
  vtt: 'videos/overview.vtt',
  youtube: '',
  drive: '',
},
```

`index.html` の目次に `#overview` セクションを追加する必要があります（未追加の場合）。

## 6. 作る動画の優先順位

---

## 7. 自動音声の品質について

| メリット | デメリット |
|----------|------------|
| 声質が一定 | やや機械っぽい |
| 台本修正→再生成が簡単 | 感情・強調は弱い |
| 収録のたびに読み上げ不要 | 画面操作とのタイミング調整は人手 |

**おすすめ:** まず自動音声で一連の動画を揃え、TFチームに確認してもらう。問題なければそのまま公開、気になる章だけ人の声で差し替え。

---

## 8. トラブルシュート

| 症状 | 対処 |
|------|------|
| `edge-tts` がない | `pip install -r requirements-narration.txt` |
| 音声が英語になる | JSON の `voice` を `ja-JP-` で始まるものに |
| 動画に音が乗らない | ffmpeg の `-map` オプションを確認 |
| 字幕がずれる | VTT を手編集、または収録ペースを台本に合わせて再収録 |
| MP4 が Vercel で重い | 章ごとに短く保つ（5分以内）、解像度720p程度 |
